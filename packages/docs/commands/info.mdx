---
title: "info"
description: "Show detailed media file information"
---

The `info` command displays detailed information about media files including format, codecs, resolution, duration, and more.

## Syntax

```bash
lazyff info <input> [options]
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `input` | Yes | Path to the media file |

## Options

| Option | Alias | Type | Default | Description |
|--------|-------|------|---------|-------------|
| `--json` | `-j` | boolean | false | Output as JSON |

## Examples

### Basic Info

```bash
lazyff info video.mp4
```

Format:    QuickTime / MOV
Duration:  2:35

Streams:   2

  Resolution: 1920x1080
  Aspect:     16:9
  Frame rate: 29.97 fps

  Bitrate:    6.5 Mbps

Audio:
  Codec:      AAC
  Sample:     48000 Hz
  Channels:   stereo
  Bitrate:    256 kbps

## Information Displayed

### Format Section

| Field | Description |
|-------|-------------|
| File | Filename |
| Format | Container format |
| Duration | Total duration (HH:MM:SS) |
| Size | File size |
| Bitrate | Overall bitrate |
| Streams | Number of streams |

### Video Section

| Field | Description |
|-------|-------------|
| Codec | Video codec and profile |
| Resolution | Width x Height |
| Aspect | Display aspect ratio |
| Frame rate | Frames per second |
| Pixel fmt | Pixel format (color space) |
| Bitrate | Video stream bitrate |

### Audio Section

| Field | Description |
|-------|-------------|
| Codec | Audio codec |
| Sample | Sample rate in Hz |
| Channels | Channel layout |
| Bitrate | Audio stream bitrate |

### Subtitles Section

| Field | Description |
|-------|-------------|
| Index | Stream index |
| Codec | Subtitle format |
| Language | Language code (if available) |

## Tips

<Tip>
  Use JSON output to pipe info to other tools:
  ```bash
  lazyff info video.mp4 --json | jq '.video.resolution'
  ```
</Tip>

<Tip>
  Check video duration before trimming:
  ```bash
  lazyff info video.mp4  # Note the duration
  lazyff trim video.mp4 --start 00:01:00 --end 00:02:00
  ```
</Tip>

<Tip>
  Verify video specs before uploading to platforms:
  ```bash
  lazyff info youtube_ready.mp4
  # Check: H.264, AAC, reasonable bitrate
  ```
</Tip>

## Common Use Cases

### Check Before Converting

```bash
# See current format/codec
lazyff info input.mov
# Then convert as needed
lazyff convert input.mov output.mp4
```

### Verify Compression Results

```bash
# Before
lazyff info original.mp4
# Size: 500 MB, Bitrate: 15 Mbps

lazyff compress original.mp4 --target-size 100MB

# After
lazyff info original_compressed.mp4
# Size: 98 MB, Bitrate: 3 Mbps
```

### Debug Merge Issues

```bash
# Check if files are compatible
lazyff info part1.mp4
lazyff info part2.mp4
# Compare codecs, resolution, fps
```

### Batch Info (Shell)

```bash
# Get info for all MP4 files
for f in *.mp4; do echo "=== $f ==="; lazyff info "$f"; done
```
