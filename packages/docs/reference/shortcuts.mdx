Complete reference for keyboard shortcuts in ffwrap TUI mode.

## Global Shortcuts

These shortcuts work anywhere in the TUI:

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Exit ffwrap |
| `Ctrl+P` | Open help dialog |
| `Esc` | Close current dialog / Cancel operation |

## Prompt Shortcuts

When the prompt is focused:

### Navigation

| Shortcut | Action |
|----------|--------|
| `←` / `→` | Move cursor left/right |
| `Home` | Move to beginning of line |
| `End` | Move to end of line |

### Editing

| Shortcut | Action |
|----------|--------|
| `Backspace` | Delete character before cursor |
| `Delete` | Delete character after cursor |
| `Ctrl+Backspace` | Delete word before cursor |
| `Ctrl+U` | Clear entire line |
| `Esc` | Clear entire line |

### History

| Shortcut | Action |
|----------|--------|
| `↑` | Previous command in history |
| `↓` | Next command in history |

### Autocomplete

| Shortcut | Action |
|----------|--------|
| `/` | Open command autocomplete (at start of input) |
| `@` | Open file autocomplete |
| `↑` / `↓` | Navigate suggestions (when open) |
| `Enter` | Accept selected suggestion |
| `Esc` | Close suggestions |

### Execution

| Shortcut | Action |
|----------|--------|
| `Enter` | Execute current command |

## Dialog Shortcuts

When a dialog is open:

### General

| Shortcut | Action |
|----------|--------|
| `Esc` | Close dialog / Cancel |
| `Enter` | Confirm / Select |

### Selection Dialog

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate options |
| `Enter` | Select highlighted option |
| `j` / `k` | Navigate options (vim-style) |

### Confirmation Dialog

| Shortcut | Action |
|----------|--------|
| `Enter` | Confirm (Yes) |
| `Esc` | Cancel (No) |

## File Browser Shortcuts

When the file browser/sidebar is open (`Ctrl+B` to toggle):

### Navigation

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate files |
| `Enter` | Select file / Enter directory |
| `Backspace` | Go to parent directory |
| `Home` | Go to first item |
| `End` | Go to last item |
| `PageUp` | Page up |
| `PageDown` | Page down |

### Vim-style Navigation

| Shortcut | Action |
|----------|--------|
| `j` | Move down |
| `k` | Move up |
| `h` | Go to parent directory |
| `l` | Enter directory / Select file |

## Quick Reference Card

### Most Used

| Shortcut | Action |
|----------|--------|
| `Enter` | Execute / Confirm / Select |
| `↑` / `↓` | Navigate |
| `Esc` | Cancel / Close / Clear |
| `Ctrl+P` | Help |
| `Ctrl+C` | Exit |

### Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+U` | Clear line |
| `Ctrl+Backspace` | Delete word |
| `Home` | Start of line |
| `End` | End of line |

### Panels

| Shortcut | Action |
|----------|--------|
| `Ctrl+B` | Toggle file browser |
| `Tab` | Cycle focus between panels |

## Terminal Compatibility

Some shortcuts may behave differently depending on your terminal emulator:

| Terminal | Notes |
|----------|-------|
| iTerm2 | Full support |
| Terminal.app | Full support |
| Windows Terminal | Full support |
| Alacritty | Full support |
| Kitty | Full support |
| GNOME Terminal | Full support |
| Konsole | Full support |

<Note>
  If shortcuts aren't working, ensure your terminal isn't intercepting them. Check your terminal's keyboard settings.
</Note>

## Tips

<Tip>
  Learn these shortcuts first: `↑` (history), `Ctrl+P` (help), and `Ctrl+U` (clear line). They'll make you productive quickly.
</Tip>

<Tip>
  Vim users: `j`/`k` navigation works in dialogs and the file browser.
</Tip>

<Tip>
  Use `Ctrl+Backspace` to quickly delete a word, or `Ctrl+U` to clear the entire line and start over.
</Tip>
