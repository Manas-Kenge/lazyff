Complete reference for keyboard shortcuts in ffwrap TUI mode.

## Global Shortcuts

These shortcuts work anywhere in the TUI:

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Exit ffwrap |
| `Ctrl+P` | Open help dialog |
| `Esc` | Close current dialog / Cancel operation |

## Command Shortcuts (with a file selected)

When a file is selected in the file tree, these shortcuts trigger operations:

| Shortcut | Action |
|----------|--------|
| `i` | Show media info |
| `c` | Open Compress dialog |
| `v` | Open Convert dialog |
| `e` | Open Extract dialog |
| `g` | Open GIF dialog |
| `r` | Open Trim dialog |
| `t` | Open Thumbnail dialog |
| `m` | Open Merge dialog |

## Dialog Shortcuts

When a dialog is open:

### General

| Shortcut | Action |
|----------|--------|
| `Esc` | Close dialog / Cancel |
| `Enter` | Confirm / Select |

### Selection Dialog

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate options |
| `Enter` | Select highlighted option |
| `j` / `k` | Navigate options (vim-style) |

### Confirmation Dialog

| Shortcut | Action |
|----------|--------|
| `Enter` | Confirm (Yes) |
| `Esc` | Cancel (No) |

## File Browser Shortcuts

When the file browser/sidebar is open (`Ctrl+B` to toggle):

### Navigation

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate files |
| `Enter` | Select file / Enter directory |
| `Backspace` | Go to parent directory |
| `Home` | Go to first item |
| `End` | Go to last item |
| `PageUp` | Page up |
| `PageDown` | Page down |

### Vim-style Navigation

| Shortcut | Action |
|----------|--------|
| `j` | Move down |
| `k` | Move up |
| `h` | Go to parent directory |
| `l` | Enter directory / Select file |

## Terminal Compatibility

Some shortcuts may behave differently depending on your terminal emulator:

| Terminal | Notes |
|----------|-------|
| iTerm2 | Full support |
| Terminal.app | Full support |
| Windows Terminal | Full support |
| Alacritty | Full support |
| Kitty | Full support |
| GNOME Terminal | Full support |
| Konsole | Full support |

<Note>
  If shortcuts aren't working, ensure your terminal isn't intercepting them. Check your terminal's keyboard settings.
</Note>

## Tips

<Tip>
  Learn these shortcuts first: `↑` (history), `Ctrl+P` (help), and `Ctrl+U` (clear line). They'll make you productive quickly.
</Tip>

<Tip>
  Vim users: `j`/`k` navigation works in dialogs and the file browser.
</Tip>

<Tip>
  Use `Ctrl+Backspace` to quickly delete a word, or `Ctrl+U` to clear the entire line and start over.
</Tip>
