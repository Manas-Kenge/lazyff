## What is lazyff?

lazyff is a command-line tool that makes working with ffmpeg simple and intuitive. Instead of memorizing complex ffmpeg commands, you can use straightforward commands with sensible defaults.

**lazyff provides:**

- **Simple CLI commands** - Convert, compress, trim, merge, and more with easy-to-remember syntax
- **Interactive TUI** - A beautiful terminal interface for guided media operations
- **Smart defaults** - Optimal presets for quality, formats, and codecs
- **Helpful errors** - Clear error messages with actionable suggestions

## Quick Example

```bash
# Convert a video to MP4
lazyff convert video.mov video.mp4

# Compress to a target file size
lazyff compress video.mp4 --target-size 25MB

# Trim a clip
lazyff trim video.mp4 --start 00:01:00 --duration 30

# Create a GIF
lazyff gif video.mp4 --start 5 --duration 3 --width 480
```

## Two Ways to Use

### CLI Mode

Run commands directly from your terminal:

```bash
lazyff convert input.mov output.mp4 --quality high
```

### TUI Mode

Launch the interactive terminal interface:

```bash
lazyff
```

## Requirements

- **Bun** (or Node.js) - JavaScript runtime
- **ffmpeg** - Must be installed and available in your PATH

<Note>
  lazyff is a wrapper around ffmpeg. All the heavy lifting is done by ffmpeg - lazyff just makes it easier to use.
</Note>
