## Prerequisites

To use ffwrap, you'll need:

### A Modern Terminal (for TUI)

For the best experience with ffwrap's interactive TUI, use a modern terminal emulator:

- [WezTerm](https://wezfurlong.org/wezterm/) (cross-platform)
- [Alacritty](https://alacritty.org/) (cross-platform)
- [Ghostty](https://ghostty.org/) (Linux, macOS)
- [Kitty](https://sw.kovidgoyal.net/kitty/) (Linux, macOS)
- [Windows Terminal](https://aka.ms/terminal) (Windows)
- iTerm2 (macOS)

<Note>
  CLI commands work in any terminal. The modern terminal recommendation is specifically for the interactive TUI mode.
</Note>

### ffmpeg

ffwrap requires ffmpeg for media processing.

<Tabs>
  <Tab title="macOS">
    ```bash
    brew install ffmpeg
    ```
  </Tab>
  <Tab title="Linux">
    ```bash
    # Ubuntu/Debian
    sudo apt update && sudo apt install ffmpeg

    # Fedora
    sudo dnf install ffmpeg
    ```
  </Tab>
  <Tab title="Windows">
    ```powershell
    # Using Chocolatey
    choco install ffmpeg

    # Or using Scoop
    scoop install ffmpeg
    ```
  </Tab>
</Tabs>

## Install ffwrap

<Tabs>
  <Tab title="npm">
    ```bash
    npm install -g @ffwrap/cli
    ```
  </Tab>
  <Tab title="bun">
    ```bash
    bun install -g @ffwrap/cli
    ```
  </Tab>
  <Tab title="pnpm">
    ```bash
    pnpm add -g @ffwrap/cli
    ```
  </Tab>
  <Tab title="yarn">
    ```bash
    yarn global add @ffwrap/cli
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
ffwrap version

# Should output:
# ffwrap: 0.0.1
# ffmpeg: 6.x.x (your version)
```

## Launch the TUI

Run ffwrap without any arguments to open the interactive interface:

```bash
ffwrap
```

Use `/` to see available commands, `@` to select files, and `Ctrl+P` for help.

## Quick CLI Examples

<AccordionGroup>
  <Accordion title="Convert video format">
    ```bash
    # Convert MOV to MP4
    ffwrap convert video.mov video.mp4

    # Convert to WebM for web
    ffwrap convert video.mp4 -f webm
    ```
  </Accordion>
  <Accordion title="Compress video">
    ```bash
    # Compress to target file size
    ffwrap compress video.mp4 --target-size 25MB

    # Compress to 50% of original
    ffwrap compress video.mp4 --percent 50
    ```
  </Accordion>
  <Accordion title="Trim video">
    ```bash
    # Cut from 1:00 to 2:00
    ffwrap trim video.mp4 --start 00:01:00 --end 00:02:00

    # Extract 30 seconds starting at 1 minute
    ffwrap trim video.mp4 --start 00:01:00 --duration 30
    ```
  </Accordion>
  <Accordion title="Create GIF">
    ```bash
    # Create GIF from video
    ffwrap gif video.mp4

    # Create 3-second GIF starting at 5s
    ffwrap gif video.mp4 --start 5 --duration 3
    ```
  </Accordion>
</AccordionGroup>
