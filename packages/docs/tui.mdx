ffwrap includes an interactive Terminal User Interface (TUI) that provides a guided experience for media operations.

## Launching the TUI

Simply run ffwrap without any command:

```bash
ffwrap
```

## Basic Usage

### Commands

Type `/` to see available commands:

| Command | Description |
|---------|-------------|
| `/convert` | Convert to different format |
| `/compress` | Reduce file size |
| `/trim` | Cut portion of video |
| `/merge` | Concatenate multiple files |
| `/extract` | Extract audio/video/frames |
| `/gif` | Convert to animated GIF |
| `/thumbnail` | Generate thumbnail |
| `/info` | Show file information |
| `/theme` | Change color theme |
| `/help` | Show help |
| `/quit` | Exit ffwrap |

### File Selection

There are two ways to select files:

1. **File Browser**: Press `Ctrl+B` to toggle the sidebar, navigate with arrow keys, and press `Enter` to select.

2. **@ Autocomplete**: Type `@` followed by a filename to search and select files.

### Keyboard Controls

| Key | Action |
|-----|--------|
| `Enter` | Execute command / Select item |
| `↑` / `↓` | Navigate history / suggestions |
| `Ctrl+P` | Open help dialog |
| `Ctrl+B` | Toggle file browser |
| `Ctrl+U` | Clear input line |
| `Ctrl+Backspace` | Delete word |
| `Esc` | Close dialog / Clear input |
| `Ctrl+C` | Exit ffwrap |

See the full [Shortcuts](/reference/shortcuts) reference for all keyboard controls.

## Themes

The TUI supports multiple color themes. Type `/theme` to browse and preview themes with live updates.

Available themes include:
- Catppuccin
- Dracula  
- Nord
- Tokyo Night
- Gruvbox
- GitHub
- And many more...

## Tips

<Tip>
  Type `/` at the start of the prompt to see all available commands with fuzzy search.
</Tip>

<Tip>
  Press `Ctrl+P` at any time to see available commands and shortcuts.
</Tip>

<Tip>
  The TUI remembers your command history. Use `↑` to quickly repeat previous commands.
</Tip>

<Tip>
  Use `Ctrl+U` to quickly clear the prompt and start over, or `Ctrl+Backspace` to delete word by word.
</Tip>
