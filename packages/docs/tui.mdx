ffwrap includes an interactive Terminal User Interface (TUI) that provides a guided experience for media operations.

## Launching the TUI

Simply run ffwrap without any command:

```bash
ffwrap
```

## Navigation

### Keyboard Controls

| Key | Action |
|-----|--------|
| `Enter` | Execute command / Select item |
| `Tab` | Accept autocomplete suggestion |
| `↑` / `↓` | Navigate history / suggestions |
| `Ctrl+C` | Exit ffwrap |
| `?` | Open help dialog |
| `Esc` | Close dialog / Cancel |

See the full [Shortcuts](/shortcuts) reference for all keyboard controls.

## Using the TUI

### Basic Workflow

1. Launch ffwrap: `ffwrap`
2. Start typing a command (e.g., "convert")
3. Select from autocomplete suggestions
4. Enter file paths when prompted
5. Choose options from dialogs
6. Confirm and execute

### Example: Converting a Video

<Steps>
  <Step title="Launch ffwrap">
    ```bash
    ffwrap
    ```
  </Step>
  <Step title="Type 'convert'">
    Start typing and select from autocomplete
  </Step>
  <Step title="Enter input file">
    Type or browse to your input file
  </Step>
  <Step title="Select format">
    Choose output format from the selection dialog
  </Step>
  <Step title="Choose quality">
    Select quality preset
  </Step>
  <Step title="Confirm">
    Review settings and confirm to start conversion
  </Step>
</Steps>

### Example: Compressing a Video

<Steps>
  <Step title="Select compress command">
    Type "compress" and press Enter
  </Step>
  <Step title="Enter input file">
    Provide the path to your video
  </Step>
  <Step title="Choose compression method">
    Select target size, bitrate, or percentage
  </Step>
  <Step title="Enter target">
    Input your target (e.g., "25MB")
  </Step>
  <Step title="Execute">
    Confirm to start compression
  </Step>
</Steps>

## Tips

<Tip>
  Use `Tab` to quickly accept autocomplete suggestions without typing the full command.
</Tip>

<Tip>
  Press `?` at any time to see available commands and shortcuts.
</Tip>

<Tip>
  The TUI remembers your command history. Use `↑` to quickly repeat previous commands.
</Tip>

## CLI vs TUI

| Feature | CLI | TUI |
|---------|-----|-----|
| Speed | Faster for known commands | Better for exploration |
| Scripting | Yes | No |
| Guidance | Help flags only | Interactive prompts |
| Learning | Steeper | Gentler |

**Use CLI when:**
- You know exactly what you want
- Scripting or automation
- Quick one-off commands

**Use TUI when:**
- Exploring features
- Learning ffwrap
- Need guided workflow
- Prefer visual interface
